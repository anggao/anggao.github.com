使用 Jackson

  • 加入jar 包：
    • jackson-annotation-2.2.3.jar
    • jackson-core-2.2.3.jar
    • jackson-databind-2.2.3.jar
  • 创建 com.fasterxml.jackson.databind.ObjectMapper
  • 调用ObjectMapper 的 writeValueAsString 方法把Java对象或集合转为json字符串.
  • Jackson 根据getter来定位Java对象的属性,而不是字段
  • 可以在getter方法上添加注解: @JsonIgnore, 在转为json对象时忽略该属性.
